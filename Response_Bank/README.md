# Response Bank

This folder stores reusable multi-sentence IELTS Speaking answers.

Use this for short speaking pieces that are longer than a single expression but smaller than a full session answer:

- Part 2 story blocks
- Part 3 opinion paragraphs
- 4-8 sentence explanations
- reusable comparison or advantage/disadvantage responses
- personal viewpoint paragraphs

Each response should keep its `source_session`, `topics`, `skills`, `related`, and `review` metadata so it can appear in the app, topic pages, and review queue.
