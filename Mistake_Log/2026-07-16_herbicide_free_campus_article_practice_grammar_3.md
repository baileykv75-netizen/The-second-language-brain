# Grammar Upgrade 3

Original:
If it is harmful for all human beings, we should stop it.

Better:
If it is harmful to people or the environment, the university should stop using it.

Explanation:
固定搭配是“be harmful to someone or something”。同时，“stop using it”比“stop it”更明确。
